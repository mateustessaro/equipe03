Shape-outside

Essa propriedade permite que voce defina uma forma para que o texto se envolva e flua ao seu redor.

junto a essa propriedade temos algumas formas basicas

O elemento que vamos aplicar a forma com propridade Shape-outside

precisa ter a propriedade float
  • inset()
  • circle()
  • ellipse()
  • polygon()

inset()

Pode ser usado para criar um retangulo/quadrado com um deslocamento opcional para o texto que o envolver
